Страница 1 - http://selenium1py.pythonanywhere.com/ru/catalogue/

- Пользовательский сценарий 1.1 - Добавление товара в корзину

-- Тестовый сценарий 1.1.1

Шаги:
1. Перейти на вкладку Все товары
2. Найти товар The shellcoder's handbook
3. Нажать кнопку "Добавить в корзину"

Ожидаемый результат
Товар добавлен в корзину, сообщение:
The shellcoder's handbook был добавлен в вашу корзину.



- Пользовательский сценарий 1.2 - Удаление товара из корзины

-- Тестовый сценарий 1.2.1

Предусловие: в корзине есть хотя бы 1 товар

Шаги:
1. Перейти в корзину
2. Нажать кнопку "Удалить"

Ожидаемый результат:

Товар удален и не отображается в корзине (нельзя найти элемент)


Страница 2 - http://selenium1py.pythonanywhere.com/ru/

- Пользовательский сценарий 2.1 - Регистрация на сайте

-- Тестовый сценарий 2.1.1

Предусловие - в системе нет подобной зарегистрированной учетной записи.

Шаги
1. Нажать на  кнопку "Войти или зарегистрироваться"
2. Ввести адрес электронной почты
3. Ввести пароль
4. Повторить введение пароля
4. Нажать "Зарегистрироваться"

Ожидаемый результат:
Сообщение: Спасибо за регистрацию!


- Пользовательский сценарий 2.2 - Вход в учетную запись

-- Тестовый сценарий 2.2.1

Предусловие - в системе есть уже зарегистрированная учетная запись.

Шаги
1. Нажать на  кнопку "Войти или зарегистрироваться"
2. Ввести адрес электронной почты
3. Ввести пароль
4. Нажать "Войти"

Ожидаемый результат:

Пользователь вошел в свою учетную запись. Сообщение  <текст сообщения>



- Пользовательский сценарий 2.3 - Поиск товаров на сайте
-- Тестовый сценарий 2.3.1
Шаги:
1. В поле поиска ввести слово "book"
2. Нажать "Найти"

Ожидаемый результат:
Отображаются найденные книги, есть сообщение "Найдено n результатов".
